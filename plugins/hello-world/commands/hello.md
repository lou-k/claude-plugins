# /hello

Greet the user with a friendly hello message.

## Usage

```
/hello [name]
```

## Arguments

- `name` (optional): The name to greet. Defaults to "World".

## Examples

```
/hello
```
→ "Hello, World! 👋"

```
/hello Claude
```
→ "Hello, Claude! 👋"

## Instructions

When this command is invoked:
1. Read the optional `name` argument from the user's input (default: "World").
2. Respond with: `Hello, {name}! 👋`
