# Lou's Claude Plugins

AI agents I've cooked up.

## Install the Marketplace

```
/plugin marketplace add lou-k/claude-plugins
```

## Plugins

### [Brag Doc](./brag-doc/README.md)

An agent that helps engineers write brag docs — concise, evidence-rich records of what you built, why it mattered, and what you personally contributed.

**Install:**

```
/plugin install brag-doc@lou-k
```

**Use:**

```
/brag-doc:write I just shipped a product -- let's document my accomplishments.
```

See [readme](./brag-doc/README.md) for more info.
